# Allocation System

A full-stack application designed to manage and optimize resource allocation, task assignment, and workload distribution across teams.

## 🚀 Features

- User Authentication & Authorization
- Resource Allocation Management
- Task Assignment & Tracking
- Team Workload Distribution
- Dashboard & Reporting
- Real-Time Updates
- Role-Based Access Control

## 🛠️ Tech Stack

### Frontend
- Angular / React
- TypeScript
- HTML5
- CSS3

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### DevOps
- GitHub Actions
- Docker
- CI/CD Pipeline

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/mustafagaut/allocation.git
cd allocation
```

### Install Dependencies

```bash
npm install
```

### Start Application

```bash
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
